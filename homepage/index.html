Thoughts on Progressive Enhancement

Progressive enhancement is something that focuses on the content being put out instead of the browser you're using. This makes things easier for every browser available. Essentially you are starting with a base of a website and then building upon it. Think of it as a skeleton. You want this skeleton to be a soldier for war, so you add the organs, then the muscle, then then skin, then the clothes, then the armor. That is essentially how progressive enhancement works. You work outwards so every user can have a better site experience.

I think progressive enhancement is important because it provides everyone an equal site experience. It also makes building a website easier and allows you to pass along a task to someone else if you don't know how to do it. This makes the process much more fluid and fast.
